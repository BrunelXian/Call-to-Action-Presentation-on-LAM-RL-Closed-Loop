# 一、总结

更新时间： September 24 2025
目前看到的TWI的AI/ML工作的主要内容集中在3个方向：

1.  NDT + defect detection 机器学习检测缺陷
2.  digital wein + smart manufacturing 数据驱动制造
3.  in-situ monitoring +AI analysis 原位监控数据分析

目前没有任何公开的reinforcement learning 控制研究
```
TWI现阶段停留在"AI检测/AI预测"阶段，而不是AI过程工程阶段
```

# 二、 项目总结

## 2.1 Additive Manufacturing + AI  (Lancast Uni合作)

AMIC AMIC（Additive Manufacturing Innovation Centre）
TWI 和 Lancaster University 联合建立的研究中心。

目标： 将 **AI / machine learning** 与增材制造工艺结合 [Ref](https://www.twi-global.com/innovation-network/centres/innovation-centres/additive-manufacturing-innovation-center "AMIC Ref.")

目前披露的：
- Digital Twin driven monitoring
- AM process modelling

技术路线是 
```
sensor → ML prediction
```

## 2.2 Smart Manufacturing（AI + 工艺数据）

2025 TWI 启动了一个项目：**Smart Manufacturing for Space Applications**

目标：通过 **AI 分析制造数据** 来优化工艺 [Ref](https://www.twi-global.com/media-and-events/press-releases/2025/smart-manufacturing-for-space-applications-project-launched )

技术路线是 
```
sensor → AI → insight
```

没有控制但是跟closed-loop有很多联系

## 2.3 AI in NDT（无所检测）

这个目前是TWI最成熟的AI方向

他们目前有个课题组就是 AI and Machine Learning in NDT [Ref](https://theweldinginstitute.com/insights/13447592)

技术路线是，对于pipeline 
```
NDT data → ML classifier → defect detection
```
完全是inspection的方案

## 2.4 AI + precision manufacturing（Huddersfield 合作）

Applied Artificial Intelligence Innovation Centre
2025年成立，TWI和University of Huddersfield合作

目标是  Autonomous systems + AI decision systems

规划阶段 [Ref](https://www.bindt.org/News/february-2025/university-of-huddersfield-and-twi-partner-on-strategic-rd)
